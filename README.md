# CSE160 Assignment 0 — Vector Library

Canvas-based vector visualization with an extended `Vector3` class (`lib/cuon-matrix-cse160.js`).

## Files

- `asg0.html` / `asg0.js` — main program
- `lib/cuon-matrix-cse160.js` — matrix library with vector operations

## Run locally

```bash
python3 -m http.server 8080
```

Open http://localhost:8080/asg0.html

## GitHub Pages

After pushing to `main`, enable **Settings → Pages → Deploy from branch `main` / `/ (root)`**.

Live site:

- https://jay-sangha.github.io/AS0/
- https://jay-sangha.github.io/AS0/asg0.html

## Usage

1. Enter **v1** and **v2** coordinates, then click **Draw**.
2. Choose an operation and scalar (if needed), then click the second **Draw**.
3. Open the browser console (F12) for magnitude, angle, and area results.
